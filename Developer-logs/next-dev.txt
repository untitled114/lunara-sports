### Common Issues & Solutions

#### Backend Not Responding
```bash
# Check container status
az container show --name lunara-app-backend --resource-group lunara-app-rg

# View container logs
az container logs --name lunara-app-backend --resource-group lunara-app-rg

# Restart container if needed
```


2. Verify Neon database status
3. Test connection from local environment
4. Review Django settings.production configuration
